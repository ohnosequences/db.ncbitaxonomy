    Generates a TreeMap to easily retrieve a Tree with unfold

    Generates a TreeMap to easily retrieve a Tree with unfold

    nodesFile

    path to the nodes.dmp file from the NCBI taxonomy

    namesFile

    path to the names.dmp file from the NCBI taxonomy

    returns

    Left(error) if some error ocurred when reading the lines from either nodesFile or namesFile, otherwise Right(treeMap) where treeMap contains the necessary information to compose this method with treeMapToTaxTree and get a Tree of TaxNodes

    Note

    root is the node whose parent is itself, when parsing the file, if such node is found, we will end up with Some(taxNode) in the root, otherwise root will be None, and our Tree an EmptyTree
